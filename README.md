### DDD - KAMLY 

Logiciel de gestion pour plombier chauffagiste

# Installation:
 - Clone project
 - Use node 24.13.0
 - Run npm ci
 - Run npm run test 

Il n'y a pas la logique controller pour le moment ni de BDD
On peut juste créer une intervention.

# Glossaire métier : 

- Intervention (fr/en) = RDV chez un client où un plombier ou employé réalise des travaux / un devis
- Devis = quotation (en) = document émis qui propose une intervention et son prix
- Client facturé = billable client (en)
- Client final = final client (en) = client chez qui l'intervention se déroule
